# Hockey Fantasy

A comprehensive full-stack fantasy hockey application where users act as General Managers to build and manage their dream teams.

## Overview

Hockey Fantasy allows users to compete against friends or global players by selecting real NHL players for their fantasy squad. Points are awarded based on real-life performance.

## Features

### Team Management
- **Draft & Trade**: Build your team within a starting salary cap of $2M.
- **Roster Control**: Manage lineups.

### Leagues & Competition
- **Private Leagues**: Create or join private leagues with passcodes.
- **Global Standings**: Compete on the global leaderboard.

### Real-Time Data
- **Live Scoring**: Updates based on real NHL match outcomes.
- **Player Stats**: Detailed tracking of goals, assists, and performance trends.

## Tech Stack

- **Frontend**: React, TypeScript, Vite, TailwindCSS
- **Backend**: Node.js, Express
- **Database**: PostgreSQL
- **Security**: JWT Authentication, Bcrypt password hashing
